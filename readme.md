Zenflix supports 2 types of folder structure for movies and shows. 

For shows you must put each show in its folder, the name of the folder will be the name of the show, within this folder you can either have season 1 of the show or you can have subfolders within the main show folder for each episode. 

Example: 

TV SHOWS (seasoned)
├── expanse
│   ├── Season 1
│   │   └── Episode 1
│   │   └── Episode 2
├── Season 2
│   │   └── Episode 1
│   │   └── Episode 2
├── breaking bad -- (season 1 episodes only)
│   └── Episode 1 

For movies you may create collection folders, or store movies in the default movies folder as shown below. 

MOVIES
├── sci-fi -- collection
│   └── Interstellar
│   └── 2001: A Space Odyssey
|
│──Inception -- individual movies
│──The Matrix
│──The Matrix Reloaded
│──The Matrix Revolutions
│──Blade Runner 2049
